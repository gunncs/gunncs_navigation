What is gunncs_navigation?

A ROS package for the iARoC competition. gunncs_navigation uses Microsoft Kinect sensor and iRobot Create, to navigate a PVC pipe maze.

What are you working on?

April 29, 2012

Currently we are using the Microsoft Kinect's Depth camera to produce a fake laser scan. With the laser scan we are implementing simple feature detection, for highly continuous maze traveling.Our modded iRobot Create assembly is named old-subdivision. Old subdivision features rubber bumpers, a sonar array, Microsoft Kinect, and an expanded 1/8" smoked polycarbonate cargo bay. In the works is 12V regulation for the Kinect.
